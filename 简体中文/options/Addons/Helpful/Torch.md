# <font color=#eee5be>🔥 <b>火炬</b></font> <Badge text="Helpful" type="tip" vertical="middle"/>

***

只有<font color=#8cffff>船员阵营</font>会成为火炬，他们将拥有最大的视野，且不会受到照明破坏的影响。

- 最大数量
  - 设置游戏中火炬的最大数量
- 出现概率
  - 设置游戏中火炬的出现概率
- 火炬的视野
  - 设置火炬的视野范围

:::tip 有趣的事实
火炬在Build 3.0.0 Dev 1中被称为执灯人，在Build 3.0.0 Dev 2中改名为了火炬！
:::
